# 紫微时域 GitHub 备用演示站

这个仓库发布可直接在浏览器打开的静态备用版本：

https://boboloveseggs.github.io/ziwei-time-domain-demo-pages/

当前版本包含出生日期时间输入、浏览器内真实紫微排盘、个人中心、超级详细 AI 大报告申请，以及专业版与大白话版完整报告样例。申请时需在专业版与大白话版中二选一，所选单份报告最高可长达20页。

命盘排盘仍在浏览器本机完成。只有用户在看完命盘后主动申请超级详细 AI 大报告并完成邮箱验证，邮箱和出生资料才会提交到私密报告服务；报告完成后需使用随机限时链接和邮件提取码查看，交付30天后自动删除。
